About Yenglihs

Yenglihs is a fun "language" to transform words with two modes: modern yenglihs and old yenglihs (which ignores step 7). It's not a serious language, but rather a set of simple rules to modify text for entertainment.

Note: The "English translator" doesn't work. Please use context to understand what the word is supposed to be :).

How Does Yenglihs Work?

Rules for transforming to Yenglihs:

  1. Replace the first letter with "y" (e.g., Hello → Yello)
  2. If a word begins with "y", add prefix "la-" (e.g., Yellow → Layellow)
  3. Swap the last two letters (e.g., Layellow → Layellwo)
  4. For two-letter words: swap letters and add "l" if ends in vowel (e.g., As → Sal)
  5. For single-letter vowels: add "l" (e.g., A → Al)
  6. Add "l" at the end of words ending in a vowel (e.g., Me → Mel)
  7. Modern only: Append original first letter with "y" if needed (e.g., Apple → Ypplea → Yppleya)

How to Reverse Yenglihs?

To reverse Yenglihs back to English:

  1. Remove "y" prefix or "la-" prefix
  2. Undo the last two letters swap
  3. Remove the appended first letter and any inserted "y"
  4. Restore the word to its original form
  5. For two-letter words: swap back and remove trailing "l"
  6. For single letters: remove trailing "l" if present
